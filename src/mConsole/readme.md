# mConsole
### requires
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title></title>
  </head>
  <body>
    <div class="mConsole"></div>
  </body>
</html>
```
### usage
```javascript
mConsole.log("hello"); //no css.
mConsole.log("%c%s", "color:blue;","world"); //blue colored "world"
```
